TCP-UDP - Services

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


La connexion et l'identification d'une application cliente ou serveur nécessite la mise en oeuvre d'une couche supplémentaire.

Par l'intermédiaire d'un des deux protocoles TCP ou UDP, la couche transport fournit à chaque application un numéro d'identification local unique qui pourrait s'apparenter au token de nos conversations.



Le protocole TCP offre les mêmes fonctions que le protocole UDP, mais il numérote les paquets pour en assurer la livraison dans l'ordre initial et sans perte.
