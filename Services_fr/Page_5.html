DHCP - Services

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


En tant que poste client, il est indispensable de disposer d'une adresse IP correspondant au réseau sur lequel il est connecté.

Si une opération manuelle est possible pour attribuer une adresse IP à chacun des postes, elle devient fastidieuse si leur nombre est conséquent, périlleuse pour garantir une allocation sans adresse dupliquée, voire impossible dans un cadre de mobilité.

Le service DHCP permet d'attribuer dynamiquement une adresse IP à chaque poste qui en fait la demande auprès d'un serveur local spécifiquement configuré pour lui communiquer les paramètres dont il a besoin : adresse IP, masque, adresse de passerelle et de serveur DNS.

Question
 Quel type de requête envoie un client DHCP lors de la séquence de découverte ?
Une requête en unicast
Questionner le poste en vue d’obtenir son adresse Ethernet nécessiterait d’en disposer préalablement.
Une requête en multicast
La requête a pour objet de questionner tous les postes du réseau local et non une partie de ceux qui souhaiteraient être concernés.
Une requête en broadcast
La requête a pour objet de questionner tous les postes du réseau local.
