Choisir un itinéraire - Routage entre réseaux

Mise en contexte  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
Le principe du routage est comparable à celui d'un plan routier.



Depuis une origine jusqu'à une destination, il s'agit à chaque intersection d'emprunter la route que les panneaux indicateurs désignent comme l'itinéraire à emprunter jusqu'au prochain carrefour.



Les indications constituent un ensemble de routes regroupées dans une table de routage. Chaque itinéraire est défini de façon à correspondre au mieux à la réalité. Occasionnellement, des déviations peuvent être mises en oeuvre manuellement.



Dans le cas des réseaux de petites infrastructures stables, telles que des entreprises, le routage sera pour partie statique.

Par contre, dans le cas des fournisseurs d'accès internet, le routage assurant l'interconnexion des réseaux sera dynamique afin de garantir la redondance automatique des liaisons.



Le routage dynamique s'appuie sur les mêmes principes que le routage statique, mais les routes sont calculées automatiquement en fonction de l'état des liaisons et non plus modifiées manuellement.

Le routage dynamique n'entre pas dans le cadre de ce cours.
