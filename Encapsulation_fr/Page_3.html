Vie d'un message - Encapsulation des protocoles

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


Chaque protocole apporte sa contribution sur des fonctionnalités différentes et spécifiques.

A l'envoi d'un message se succèdent une série d'ajouts d'informations dans chaque nouvel entête de trame.

A la réception, ces informations permettent de prendre des décisions et sont progressivement supprimées pour ne restituer que le message initial.
